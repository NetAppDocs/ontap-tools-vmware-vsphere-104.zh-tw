= 更改ONTAP工具管理器密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用ONTAP工具管理器變更管理員密碼。

.步驟
. 從 Web 瀏覽器啟動ONTAP工具管理器： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 使用您在部署期間提供的適用於ONTAP tools for VMware vSphere管理員憑證登入。
. 選擇螢幕右上角的*管理員*圖標，然後選擇*更改密碼*。
. 在更改密碼彈出視窗中，輸入舊密碼和新密碼詳細資料。更改密碼的約束顯示在使用者介面畫面上。
. 選擇“*更改*”來實現更改。
