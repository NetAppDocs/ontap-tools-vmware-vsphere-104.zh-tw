= 重設ONTAP工具管理器密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您忘記了ONTAP工具管理器密碼，您可以使用ONTAP tools for VMware vSphere維護控制台產生的令牌重設管理員憑證。

.步驟
. 從 Web 瀏覽器啟動ONTAP工具管理器： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 在登入畫面上，選擇*重設密碼*選項。
+
若要重設管理員密碼，您需要使用ONTAP tools for VMware vSphere產生重設令牌。

+
.. 從 vCenter Server 開啟維護控制台
.. 輸入“2”選擇系統配置選項
.. 輸入“3”更改“maint”用戶密碼。


. 在變更密碼彈出視窗中，輸入密碼重設令牌、使用者名稱和新密碼詳細資料。
. 選擇*重置*來實現變更。密碼重設成功後，您可以使用新密碼登入。
