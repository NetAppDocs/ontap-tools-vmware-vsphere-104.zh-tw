= 重置應用程式用戶密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
應用程式使用者密碼用於向 vCenter Server 註冊 SRA 和 VASA Provider。

.步驟
. 從 Web 瀏覽器啟動ONTAP工具管理器： `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. 使用您在部署期間提供的適用於ONTAP tools for VMware vSphere管理員憑證登入。
. 從側邊欄中選擇*設定*。
. 在 *VASA/SRA 憑證* 畫面中，選擇 *重設密碼*。
. 提供新密碼並確認新密碼輸入。
. 選擇*重置*來實現變更。
