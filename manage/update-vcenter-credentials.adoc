= 更新 vCenter Server 憑證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用維護控制台更新 vCenter Server 執行個體憑證。

.開始之前
您需要擁有維護使用者登入憑證。

*關於此任務*

如果您在部署後變更了 vCenter Server 的憑證，則需要使用此程序更新憑證。

.步驟
. 從 vCenter Server 開啟 VASA Provider 控制台。
. 以維護使用者登入。
. 進入 `2`選擇系統配置選單。
. 進入 `8` 變更 vCenter 憑證。
