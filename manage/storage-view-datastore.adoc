= 資料儲存庫的ONTAP儲存視圖
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP tools for VMware vSphere在設定標籤中顯示資料儲存及其磁碟區的ONTAP儲存側視圖。

.步驟
. 從 vSphere 用戶端導航到資料儲存。
. 在右側窗格中選擇“*配置*”標籤。
. 選擇 * NetApp ONTAP工具* > * ONTAP儲存*。根據資料儲存類型的不同，視圖也會發生變化。請參閱下表以了解詳情：


|===


| *資料儲存類型* | *可用資訊* 


| NFS 資料存儲 | *儲存詳細資料*頁麵包含儲存後端、聚合和磁碟區資訊。  *NFS 詳細資料*頁麵包含與 NFS 資料儲存相關的資料。 


| VMFS 資料存儲 | *儲存詳細資訊*頁麵包含儲存後端、聚合、磁碟區和儲存可用區域 (SAZ) 詳細資訊。  *儲存單元詳細資料*頁麵包含儲存單元的詳細資訊。 


| vVols資料存儲 | 列出所有捲。您可以從ONTAP儲存窗格擴充或刪除儲存。基於ASA r2 系統的vVols資料儲存不支援此視圖。 
|===
