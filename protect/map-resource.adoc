= 配置資源映射
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您應該將資源對應到受保護網站和復原網站上，以便將虛擬機器設定為故障轉移到一組主機或另一組主機。

.開始之前
您應該已經連接了受保護的網站和復原網站。


NOTE: 在 VMware Live Site Recovery 中，資源可以是資源池、ESXi 主機或 vSphere 群集。

.步驟
. 登入 vCenter Server 並選擇 *Site Recovery* > *Sites*。
. 選擇受保護的網站並選擇*管理*。
. 在管理標籤中選擇*資源映射*>*新建*來建立新的資源映射。
. 在建立資源對映精靈中，執行下列操作：
+
.. 選擇*自動準備具有符合名稱的資源映射*並選擇*下一步*。
.. 選擇受保護和復原網站所需的資料中心對象，然後選擇*新增映射*。
.. 映射建立成功後選擇*下一步*。
.. 選擇先前用於建立反向映射的對象，然後選擇*完成*。




.結果
資源對應頁面顯示受保護網站資源和復原網站資源。您可以對您環境中的其他網路執行相同的步驟。
