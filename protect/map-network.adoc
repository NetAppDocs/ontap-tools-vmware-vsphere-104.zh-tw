= 設定網路映射
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您應該在兩個站點上配置資源映射，例如 VM 網路、ESXi 主機和資料夾，以便將受保護站點中的每個資源映射到復原站點上的相應資源。

您應該完成以下資源配置：

* 網路映射
* 資料夾映射
* 資源映射
* 佔位資料存儲


.開始之前
您應該已經連接了受保護的網站和復原網站。

.步驟
. 登入 vCenter Server 並選擇 *Site Recovery* > *Sites*。
. 選擇受保護的網站並選擇*管理*。
. 在管理標籤中選擇*網路映射*>*新建*來建立新的網路映射。
. 在建立網路映射精靈中，執行下列操作：
+
.. 選擇*自動準備具有匹配名稱的網路映射*並選擇*下一步*。
.. 選擇受保護和復原網站所需的資料中心對象，然後選擇*新增映射*。
.. 映射建立成功後選擇*下一步*。
.. 選擇先前用於建立反向映射的對象，然後選擇*完成*。




.結果
網路對映頁面顯示受保護網站資源和復原網站資源。您可以對您環境中的其他網路執行相同的步驟。
