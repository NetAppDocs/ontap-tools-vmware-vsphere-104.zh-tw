= 適用於ONTAP tools for VMware vSphere的高可用性
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP tools for VMware vSphere支援高可用性 (HA) 配置，有助於在發生故障時提供適用ONTAP tools for VMware vSphere的不間斷功能。

高可用性 (HA) 解決方案可從以下原因導致的中斷中快速恢復：

* 主機故障
+

NOTE: 僅支援單節點故障。

* 網路故障
* 虛擬機器故障（Guest OS故障）
* 應用程式（ONTAP工具）崩潰


無需對適用ONTAP tools for VMware vSphere進行額外配置即可提供高可用性 (HA)。


NOTE: ONTAP tools for VMware vSphere不支援 vCenter HA。

若要啟用 HA 功能，應在部署期間或稍後在ONTAP tools for VMware vSphere VM 設定中啟用 CPU 熱添加和記憶體熱插拔。
